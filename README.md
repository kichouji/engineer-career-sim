# SIer新人エンジニア育成録

SIer（システムインテグレーター）に配属された新卒エンジニアとして、定年までのキャリアを生き抜くエンジニア人生シミュレーションゲームです。

## 🎮 ライブデモ
**[ここをクリックしてプレイ](https://kichouji.github.io/engineer-career-sim/)**

---

## 🚀 ゲームの概要
あなたは現代のSIerに配属された新人エンジニアです。Javaの基礎やExcel方眼紙の洗礼を受けながら、技術力、業務理解、コミュニケーション能力を磨き、最高のエンジニアキャリアを目指してください。

### 特徴
- **リアルなキャリア形成**: 学習、業務、交流、休息のバランスを取りながら、理想のエンジニア像を追求します。
- **市場価値システム**: あなたのスキルや実績に応じて市場価値が変動。転職活動を行うことで、市場価値に見合った年収を手にすることができます。
- **多彩なイベント**: 案件の炎上、社内勉強会、謎のヘッドハンターからの誘いなど、SIerエンジニアなら「あるある」と頷けるイベントが盛りだくさん。
- **ビジュアル体験**: AIエージェントとの共創により、没入感のあるアニメ調のイラストとモダンなUIを実現しました。

## 🛠️ 技術スタック
- **Frontend**: React 19, TypeScript
- **Build Tool**: Vite
- **Styling**: Vanilla CSS (Custom Properties)
- **Deployment**: GitHub Actions + GitHub Pages

## 📦 ローカルでの実行方法

1. リポジトリをクローン
```bash
git clone https://github.com/kichouji/engineer-career-sim.git
```

2. 依存関係のインストール
```bash
npm install
```

3. 開発サーバーの起動
```bash
npm run dev
```
