# エンジニア育成ゲーム 技術仕様書

## 1. 全体概要

### 1.1 アプリケーション形態

* **ブラウザゲーム（SPA）**
* インストール不要
* PCブラウザを主対象

### 1.2 技術スタック

* 言語：JavaScript / TypeScript（TypeScript推奨）
* フレームワーク：React
* 状態管理：React標準（useState / useReducer）
* データ保存：**なし（リロードで初期化）**

---

## 2. アーキテクチャ方針

### 2.1 設計方針

* **完全クライアントサイド**
* サーバ通信なし
* ゲーム状態はすべてメモリ上で管理
* 「1プレイ完結型」を前提

### 2.2 状態管理戦略

* グローバル状態：`useReducer`
* UI状態：`useState`
* 不変データ（イベント定義など）：定数ファイル

---

## 3. 画面構成

### 3.1 レイアウト概要

```
+--------------------------------------+
|  左ペイン          |  右ペイン        |
|--------------------|------------------|
|  イベント画像      |  ステータス一覧  |
|                    |------------------|
|  イベントテキスト  |  行動選択肢      |
|                    |------------------|
|                    |  年数・年収表示  |
+--------------------------------------+
```

### 3.2 左ペイン仕様

* 内容

  * イベント用画像（イラスト or プレースホルダ）
  * イベント説明テキスト

* 更新タイミング

  * ターン開始時
  * イベント発生時
  * エンディング時

### 3.3 右ペイン仕様

* 表示要素

  * ステータス一覧
  * 年収
  * 経過年数（年・ターン）
  * 行動選択ボタン


* 操作

  * ボタンクリックのみ（キーボード操作不要）
