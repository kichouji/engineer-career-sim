# エンジニア育成ゲーム 要件定義書

## 1. 概要

### 1.1 ゲーム名

* **「SIer新人エンジニア育成録」**

  * ※正式名称は後決定

### 1.2 ゲームジャンル

* シミュレーション／育成ゲーム
* ターン制（1ターン＝一定期間）

### 1.3 コンセプト

* 文系卒・未経験でSIerに就職した新人エンジニアが
  **勉強・経験・人間関係・キャリア選択**を通じて成長し、
  **年収1000万円エンジニアを目指す現実寄り育成ゲーム**

### 1.4 想定プレイヤー

* エンジニア志望者
* 現役エンジニア（自虐・共感目的）
* IT業界に興味のある一般ユーザー

---

## 2. 世界観・設定

### 2.1 主人公

* 文系卒
* SIerに新卒入社した新人エンジニア
* 初期状態では技術力ほぼゼロ
* 年齢は22歳想定（固定）

### 2.2 時間軸

* 1ターン＝3か月（四半期）
* 1年＝4ターン
* ゴール達成年数を評価に使用

---

## 3. ゲームの目的・ゴール

### 3.1 最終ゴール

* **年収1000万円を達成すること**

### 3.2 クリア条件

* 年収が1000万円以上になった時点でゲームクリア

### 3.3 評価ランク

* クリア時の経過年数でランク判定

| 年数     | ランク |
| ------ | --- |
| 〜7年    | S   |
| 8〜9年   | A   |
| 10〜11年 | B   |
| 12年以上  | C   |

※バランス調整前提で変更可能

---

## 4. ステータス設計

### 4.1 基本ステータス

| ステータス名 | 内容                   |
| ------ | -------------------- |
| 技術力    | プログラミング・設計・インフラなど総合力 |
| 業務理解   | 要件理解、業務知識、SIer適性     |
| コミュ力   | 顧客対応、社内調整、人脈形成       |
| 体力     | 残業耐性・炎上時の生存力         |
| メンタル   | ストレス耐性、燃え尽き防止        |
| 市場価値   | 転職・年収アップに影響          |

### 4.2 年収

* 数値として独立管理
* ステータス・職種・イベントにより増減

---

## 5. 行動システム

### 5.1 ターンごとの行動

* 各ターンで**1〜2個の行動**を選択可能

### 5.2 主な行動カテゴリ

#### 勉強

* 技術書を読む
* 資格勉強
* 個人開発
* 英語学習

→ 技術力・市場価値が上昇
→ 体力・メンタル消費あり

#### 業務

* 案件に全力投入
* 最低限でやり過ごす
* 炎上対応

→ 業務理解・年収に影響
→ 炎上時は体力・メンタル大幅減

#### 交流

* 社内勉強会
* エンジニア交流会
* SNS発信

→ コミュ力・市場価値上昇
→ 転職イベント発生率アップ

#### 休養

* 有給取得
* 何もしない

→ 体力・メンタル回復
→ 成長は遅れる

---

## 6. イベントシステム

### 6.1 イベント発生概要

* 各ターンで確率発生
* ステータスや行動により発生率変動

### 6.2 代表的イベント例

#### 業務系

* 案件炎上
* 仕様変更地獄
* 有能な先輩に出会う
* 無茶振りPM登場

#### キャリア系

* 転職の誘い
* ヘッドハンティング
* 社内異動
* フリーランス独立の誘惑

#### メンタル系

* 燃え尽き症候群
* モチベーション爆上がり
* ブラック案件確定

---

## 7. キャリア分岐

### 7.1 主なキャリアパス

* SIer上流工程特化
* Web系エンジニア転向
* マネジメント志向
* フリーランス
* 社内SE

### 7.2 分岐条件

* 特定ステータス到達
* 特定イベント選択
* 年数経過

---

## 8. ゲームオーバー条件（任意）

* メンタルが0になる → 社会逃避エンド
* 体力が0になる → 過労死エンド
* 年齢◯歳で年収1000万未達 → ノーマルエンド

---

## 9. UI・演出（概要）

* ステータス画面
* 年収・年数表示
* イベントテキスト、イメージ画像
* 軽い自虐・業界ネタを含む文章表現
